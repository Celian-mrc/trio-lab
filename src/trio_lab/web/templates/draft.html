Simulateur de draft

Simule un draft complet façon champ select : verrouille les picks des deux équipes, rôle par rôle. Le rôle actif est surligné à gauche/droite ; la grille ci-dessous liste tous les champions ayant au moins une game jouée dans ce rôle sur la fenêtre et la région choisies (pas de WR inventé sur 0 game — sur une seule région et un rôle inhabituel, le roster peut être incomplet ; passer sur « toutes régions » donne la couverture la plus large), triés par valeur ajoutée par rapport à ce qui est déjà posé — synergie avec les alliés verrouillés + contre l'adversaire même rôle s'il est déjà connu — puis par fiabilité (les échantillons trop faibles pour être lus comme un signal restent visibles, mais jamais devant un champion mieux établi). Les {{ draft_recommended_count }} premiers sont marqués « Recommandé » ; le reste du roster reste cliquable, rien n'est masqué. État entièrement dans l'URL : partage le lien pour reprendre où tu en es.

Blind pick : tant qu'aucun adversaire même rôle n'est verrouillé, le tri ne porte que sur la synergie côté allié (pas de contre à calculer) — la grille affiche alors, pour chaque champion, le NOMBRE de contres notables connus (delta ≤ 3 pts de WR contre lui) et le pire d'entre eux. Un champion avec 0 contre notable encaisse mal aucun contre fort ; un champion avec plusieurs contres notables est un blind pick plus risqué, même si son pire cas n'est pas extrême.
